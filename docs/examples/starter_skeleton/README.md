# 下一阶段 Starter Skeleton

本目录用于提供“下一阶段重构”的样板骨架。

这些文件当前不参与现有原型运行，也不会影响 `run_demo.py`、`pytest` 和现有 `src/heeg_auto` 结构。

它们的作用是：

- 给团队说明后续目录和职责应该怎么拆
- 给“新建患者”模块提供第一份可照着搬运的样板
- 把“动作 / 元素 / 子模块 / 大模块 / 用例 / pytest 接入”用具体文件落下来

建议使用方式：

1. 先继续保留当前可运行原型
2. 以本目录为参考，在真实 `src/heeg_auto` 中逐步迁移
3. 优先从 `新建患者` 模块开始，验证整套新结构是否顺畅

本目录当前包含：

- `elements/`：元素定义样板
- `submodules/`：子模块样板
- `modules/`：大模块样板
- `cases/`：正式用例样板
- `runner/`：执行桥接样板
- `tests/`：pytest 接入样板
