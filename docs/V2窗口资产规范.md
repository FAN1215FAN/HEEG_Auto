# V2窗口资产规范

## 目标

窗口资产用于定义步骤中的 `窗口` 上下文，承担窗口级稳定命名和基础锚点治理。

## 推荐字段

```yaml
窗口资产:
  - 窗口标识:
    资产类型: 窗口
    中文名称:
    所属窗口:
    AutomationId:
    ControlType:
    Name:
    ClassName:
    是否唯一:
    锚点元素:
    用途说明:
```

## 约束

- `中文名称` 必须稳定且唯一。
- 业务 case 中统一引用 `中文名称`，不要直接引用窗口标识。
- 同一个业务窗口不要同时出现多个名字。
- `锚点元素` 用于说明后续稳定定位依赖，不是强制字段。

## 当前项目窗口

- 患者主界面
- 患者列表
- 创建患者
- 创建设备检查
- 模拟器确认窗口
- 设备设置

- 阻抗窗口

## 当前重点缺口

- 部分窗口还缺稳定锚点元素定义。
- 新窗口接入时必须先补窗口资产，再写 case。
