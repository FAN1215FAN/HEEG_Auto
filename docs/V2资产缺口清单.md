# V2资产缺口清单

## 当前缺口

- `设备设置 -> 脑电工作模式`
  - 目前缺 `AutomationId`
  - 建议研发补充稳定标识

- `创建患者 -> 标题栏关闭按钮`
  - 当前无稳定锚点
  - 后续建议单独补 `AutomationId`

- `患者列表第一行 / 检查记录行 / 采集记录行`
  - 目前缺稳定行级锚点
  - 双击和右键流程后续更依赖这类资产

- `保存成功 / 保存失败 / 主界面结果文本`
  - 当前结果锚点还不够丰富
  - 后续建议补更多成功/失败状态元素

## 当前建议

1. 研发优先补 `脑电工作模式` 的 `AutomationId`。
2. 研发补齐列表行和记录行稳定标识。
3. 对重复出现的通用按钮统一补 `AutomationId`，减少多窗口重名歧义。
4. 每新增一个业务窗口，先补窗口资产和断言资产，再新增 case。
