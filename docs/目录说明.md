# 目录说明

## 核心目录

- `src/heeg_auto/config`
  - 配置、定位和项目级常量
- `src/heeg_auto/core`
  - driver、基础页面能力、动作执行器、报告等基础设施
- `src/heeg_auto/pages`
  - 页面对象封装
- `src/heeg_auto/modules`
  - V1 模块资产，当前仅保留兼容
- `src/heeg_auto/cases`
  - formal case 与目录支持文件
- `src/heeg_auto/runner`
  - case 识别、执行计划、目录生命周期、suite 调度
- `src/heeg_auto/v2`
  - V2 解析器、执行器、资产加载器与资产目录
- `tests`
  - pytest 测试
- `docs`
  - 中文文档
- `artifacts`
  - 报告、截图、日志等产物
