# V2元素资产规范

## 目标

元素资产用于定义 `参数 / 元素 / 按钮` 的稳定名称和定位方式，是 V2 执行器默认动作推断的基础。

## 推荐字段

```yaml
元素资产:
  - 元素标识:
    资产类型: 元素
    中文名称:
    所属窗口:
    AutomationId:
    ControlType:
    Name:
    ClassName:
    是否唯一:
    锚点元素:
    用途说明:
```

## 命名规则

- 同一窗口内禁止两个元素都叫 `关闭`。
- 必须按业务语义命名，例如：
  - `底部关闭按钮`
  - `标题栏关闭按钮`
  - `患者列表第一行`
- `按钮` 字段引用的也是元素资产，只是语义上表示“默认点击”。

## 控件类型与默认动作

- `Button` -> 点击
- `Edit` -> 输入
- `Custom` -> 输入
- `ComboBox` -> 下拉选择
- `RadioButton` -> 选择单选
- `CheckBox` -> 设置勾选
- 其他控件若要默认推断，需要先明确是否允许直接点击

## 当前重点缺口

- 脑电工作模式缺 `AutomationId`
- 患者列表第一行缺稳定锚点
- 标题栏关闭按钮缺稳定锚点
- 更多列表行、结果行、记录行资产仍需补充
