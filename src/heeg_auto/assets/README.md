# Assets

当前正式资产目录分为 3 层：

- `windows/`
  - 窗口资产
  - 负责窗口定位与复杂图形区域的 `交互标定`
- `elements/`
  - 元素资产
  - 负责稳定控件定位
- `assertions/`
  - 断言资产
  - 负责命名断言治理

当前约束：
- 不新增“区域资产”
- 波形区和进度条类交互统一通过窗口资产内的 `交互标定` 处理
- `recommended_*.yaml` 继续保留为推荐模板，不作为业务正式资产

详细规范见：
- `docs/窗口资产规范.md`
- `docs/元素资产规范.md`
